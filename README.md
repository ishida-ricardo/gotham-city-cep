## Gotham City Cep

### Instruções para instalar e iniciar o projeto:

* Instalação:

 ```
    git clone link-do-projeto
    cd gotham-city-cep
    yarn install
    cd backend
    yarn install
 ```

* Inicialização:

 ```
    yarn start
    cd backend
    yarn nodemon src/server.js
 ```

* Testes:

 ```
    cd backend
    yarn test
 ```
