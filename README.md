Gotham City Cep

Instruções para instalar e iniciar o projeto:

* Instalação:

-- Instalar o front-end:
---- > yarn install

-- Instalar o back-end:
---- > cd backend
---- > yarn install

* Inicialização:

-- Front-end:
---- Na raiz do projeto, executar: yarn start

-- Back-end:
---- Na pasta "backend", executar: yarn nodemon src/server.js

* Testes:

-- Back-end:
---- Na pasta "backend", executar: yarn test
